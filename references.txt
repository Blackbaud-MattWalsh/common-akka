0. make sure minikube is running: `> minikube start`
1. run `> gradle build docker` to build docker image
1. create kubernetes service `> kubectl create -f kubernetes-service.json`
1. create kubernetes stateful set `> kubectl create -f kubernetes-stateful-set.json`
1. run the pod: `> kubectl scale statefulsets myapp --replicas=3`
